# Robot-Soccer
A ESP 32 Powered robot that is controlled via wifi to play soccer

Heres a wiring diagram of our robot: 
<img width="1408" height="768" alt="Wiring Diagram" src="https://github.com/user-attachments/assets/57dbe72c-b077-47cd-be58-c99ce78adba8" />

List of components:
1. ESP-32
2. Servo Motor
3. L298N Motor Driver
4. 2X OSEPP Stock DC Motors
5. Jumper Wires
6. 2X 3.7V Li-on Batteries
